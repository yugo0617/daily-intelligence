# 週刊IT・ServiceNowレポート 2026-W28

## 今週のサマリー

2026年第28週（7月13日～7月19日）におけるServiceNow、AI、クラウド、セキュリティ分野の主要な動向をまとめました。特に、ServiceNowの新機能やAI統合の進展、主要AI企業の最新動向、クラウドサービスのアップデート、重要なセキュリティインシデントに焦点を当てています。

## ServiceNow最新情報

### 製品・機能アップデート

**Now Assist Suiteの最新リリース**

ServiceNowは、Now Assist Suiteの最新バージョンをリリースしました。このリリースでは、以下のアップデートが含まれています：

- **Conversational Help 2.0.3**：Now Assistパネル内での「Get Help Skill」の自動検出機能が無効化され、ユーザーエクスペリエンスが向上しました。

- **Now Assist for Accounts Payable Operations (APO) 6.0.2**：請求書ケースの要約機能におけるマイナーな修正が行われました。

- **Now Assist for Code Generation 28.5.13**：Code Assistが新しいCode Retrievalバージョンを使用するよう更新され、セキュリティ脆弱性の修正や仮想テーブルのインデックス作成の停止などが含まれています。

- **Now Assist for Digital End-user Experiences (DEX) 4.0.0**：サービスデスクエージェントがZoomの通話品質問題を診断・解決できる新機能が追加され、デバイス、ネットワーク、アプリケーションデータを統合的に分析することで、根本原因の特定と解決策の提案が可能となりました。

これらのアップデートにより、エンジニアはより効率的に問題を特定し、迅速な対応が可能となります。

### Now Assist・AI機能

**Now Assist for ITSMの実装ガイド公開**

AIgent Labは、ServiceNowのNow AssistとAI Agent Studioを活用した業務AIエージェントの構築に関する詳細な実装ガイドを公開しました。このガイドでは、Skill Kit、Workflow Studioとの連携、AnthropicやOpenAIのモデル接続、ITSM、HRSD、CSMへの実装パターンなどが公式ドキュメントに基づいて解説されています。

この情報は、ServiceNowのAI機能を活用した業務効率化を目指すエンジニアにとって、実践的な参考資料となります。

### コミュニティ・認定資格

**Horizon Design Systemのアップデート**

ServiceNowのHorizon Design Systemにおいて、以下のアップデートが行われました：

- **AIセクションの新設**：AIデザインガイダンスが独立したセクションとして追加され、関連するデザインパターンやガイドライン、リソースが一元化されました。

- **アクセシビリティペルソナカードの導入**：新たにアクセシビリティペルソナカードが追加され、デザインプロセスにおける共感の構築や、実際のニーズに対する体験の検証が容易になりました。

- **ネイティブモバイルの強化**：ネイティブモバイル向けのコンポーネントやスクリーンが拡充され、ServiceNow上でのモバイル体験の構築がより充実しました。

これらのアップデートは、ユーザーエクスペリエンスの向上やアクセシビリティの強化に寄与し、エンジニアがより包括的なデザインを実現するための支援となります。

## AI・生成AI動向

### 主要モデル・プラットフォームの動き

**ServiceNowのAI統合の進展**

ServiceNowは、Knowledge 2026カンファレンスにおいて、AI統合の新たな進展を発表しました。特に、Moveworksの買収により、エンタープライズ全体での会話型AI、エンタープライズ検索、AI音声エージェント、データ探索機能を統合した「ServiceNow Otto」を導入しました。これにより、ユーザーは特定のシステムを意識することなく、意図を解釈し、適切なエージェントに作業をルーティングし、実行することが可能となります。

この統合は、エンタープライズAIの活用を加速させ、業務プロセスの自動化と効率化を推進します。

### エンタープライズAI活用

**ServiceNowのAI制御タワーの強化**

ServiceNowは、AI Control Towerの機能強化を発表しました。これにより、Microsoft Azure、AWS、Google Cloud Platform（GCP）などの主要なLLMプロバイダーとの統合が拡大し、SAP、Oracle、Workdayなどのアプリケーションとも連携が可能となりました。これにより、エンタープライズインフラ全体のガバナンスと可視性が向上し、AIの統合管理が容易になります。

エンジニアにとって、これらの統合は、複数のプラットフォームやアプリケーション間でのAI活用を一元的に管理し、業務プロセスの最適化を図る上で重要な役割を果たします。
